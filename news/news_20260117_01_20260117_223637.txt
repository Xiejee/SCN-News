DATE: 2026-01-17 22:37
---
大家好！很高兴宣布 SCN (Songhe Class News) 新闻发布系统已成功创建并投入使用。

【运行原理】
本程序是一个轻量级的“无后端”架构。它利用 GitHub Pages 托管静态网页，通过 JavaScript 调用 GitHub 开放 API。每当你打开网页时，它会自动“扫描”仓库文件夹中以 news_ 开头的文本文件，解析其内部的标题和时间标注，并按时间顺序进行丝滑的视觉呈现。

【使用建议】
1. 访问建议：由于本程序的数据存储在 GitHub 服务器上，在部分网络环境下访问可能会有延迟。为了获得最流畅的加载体验（尤其是在拉取最新 TXT 时），推荐使用网络加速器或代理工具。
2. 更新提示：上传新的新闻文件后，GitHub 可能需要 1-2 分钟同步。如果网页没有立即显示，请稍等片刻后刷新。
3. 格式规范：请后续发布者务必保留 TITLE、DATE 和 --- 这三个核心标记，以确保程序能正确解析。

这是 SCN 迈出的第一步，期待它能成为我们班级高效信息传递的桥梁！
